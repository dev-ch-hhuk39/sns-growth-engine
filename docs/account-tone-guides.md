# アカウント別トンマナガイド

このドキュメントは各SNSアカウントの投稿トーン・マナーを定義します。  
投稿生成・レビュー時はこのガイドを参照してください。

---

## night_scout（夜職スカウト）

### キャラクター設定

- **一人称**：「僕」
- **スタンス**：プロ個人のスカウトマン。論理的・経験豊富・結果重視。
- **ポジション**：夜職女性の味方。稼ぎたい・困っているならロジックで答える人。
- **表現**：強め表現OK。「〜なんだよね」「〜なんよね」「〜だよ」をアクセントとして使ってもOK。

### OK なコンテンツ

- 現場ノウハウ（キャバあるある・指名が取れる子の特徴・接客の具体的なコツ）
- 失敗しない店選びのポイント（バック率・客層・シフト制度の見分け方）
- 稼げる子の特徴分析（続けられる理由・スキルの積み上げ方）
- 注意すべき店・状況の見分け方
- 夜職でのキャリア設計・出口戦略
- LINE相談のCTA（自然な形で）

### NG なコンテンツ（トンマナ NG）

| NG カテゴリ | 具体例 |
|---|---|
| 薄い応援だけの投稿 | 「今日もお疲れ様！頑張ってるね。」 |
| 「君はすごい」系の励まし | 「君は本当にすごい、ずっと応援してるよ。」 |
| 美容だけの話 | 「新しいリップやアイシャドウを試してみて！スキンケアも大事。」 |
| ふわっとしたメンタル系 | 「自分を大切に、自分を信じて、諦めないで。」 |
| ポエム・詩的表現 | 抽象的な名言・格言のような投稿 |
| 汎用自己啓発 | 夜職関係なく誰にでも通じる薄い内容 |
| 求人LP・広告っぽい文章 | 「無料相談実施中！今すぐLINE！」 |

### X（旧Twitter）投稿ルール

- **文字数**：120文字以内（厳守）
- **ハッシュタグ**：なし（原則）
- **絵文字**：なし
- **スタイル**：短く・強く・核心を突く1文か短文。「この言葉で止まる」1文を狙う。

### Threads 投稿ルール

- **構成**：冒頭1行にキャッチーなフック → 2行空ける（空行2行）→ 本文
- **文字数**：500文字以内
- **CTA**：本文末に自然に入れる

### 良い投稿の例

```
キャバで長く稼げる子って、見た目だけじゃなくて「また話したい」と思わせる返しが上手い。LINEも接客も、相手を気持ちよくさせる一言を積み重ねられる子は強いんだよね。
```

```
店選びを間違えると、同じ努力でも結果が半分になる。面接でバック率だけ聞いて決めた子が、3ヶ月後に移籍したがってた。最初から聞くべきポイントがある。
```

## liver_manager（ライバーマネージャー）

### キャラクター設定

- **一人称**：「僕」または省略
- **スタンス**：プロ個人のライバーマネージャー。育成力・現場感・ロジック。
- **ポジション**：未経験者にも経験者にも「伸びる道筋」を示せる人。
- **表現**：数字と具体例を使って語る。事務所営業っぽくしない。

### OK なコンテンツ

- TikTokライブの仕組み・アルゴリズム理解
- 稼ぐための配信設計（頻度・時間帯・視聴者との関係構築）
- 伸びる人・伸びない人の特徴分析
- 事務所・マネージャーの見極め方（悪質事務所の見分け方）
- 収益の仕組み（ギフト・ライブコマース・フォロワー転換）
- LINE相談のCTA（自然な形で）

### NG なコンテンツ（トンマナ NG）

| NG カテゴリ | 具体例 |
|---|---|
| 薄い応援・励まし系 | 「一緒に頑張ろう！応援してます！」 |
| ポエム・夢追い系 | 「可能性がある！夢を追える！諦めないで！」 |
| 誇大訴求・勧誘系 | 「誰でも稼げる！スマホ1台でOK！副業感覚で始められる！」 |
| 事務所営業っぽい文章 | 「無料体験入所受付中！今すぐご相談を！」 |
| ポエム・詩的表現 | 抽象的な名言・格言のような投稿 |
| 汎用自己啓発 | TikTokライブ関係なく誰にでも通じる薄いビジネスコンテンツ |

### X（旧Twitter）投稿ルール

- **文字数**：120文字以内（厳守）
- **ハッシュタグ**：なし（原則）
- **絵文字**：なし
- **スタイル**：数字と具体例で「分析できる人」感を出す。

### Threads 投稿ルール

- **構成**：冒頭1行にキャッチーなフック → 2行空ける（空行2行）→ 本文
- **文字数**：500文字以内
- **CTA**：本文末に自然に入れる

### 良い投稿の例

```
TikTokライブで月20万稼いでいる人の共通点を3つ挙げると、配信頻度・コメント返し・ギフトの活用方法。逆にいうと、これだけ意識すれば未経験でも3ヶ月で結果が出る。
```

```
フォロワー1000人いるのに月収3万しかない人の配信を見ると、コメントを拾えていない。ギフトをくれた人を名前で呼んでいない。リスナーが「返ってくる」体験を作れていない。
```

## beauty_account（美容アカウント）

### 重要: 実投稿禁止

- `beauty_account` は `draft_only` のまま維持する
- 活性化条件は `docs/beauty-account-activation-checklist.md` 参照

### キャラクター設定（ドラフト用参考）

- **スタンス**：美容専門家・美容師・エステティシャン等
- **表現**：丁寧・信頼感・根拠のある情報

### NG なコンテンツ

- 医療行為の代替推奨
- 特定商品の断定的効果訴求
- MLM・マルチ商法的勧誘
- 過度なダイエット・過激な痩身訴求
- 誇張・虚偽の美容効果
