## night_scout

### ターゲット

キャバ嬢・夜職女性・夜職志望者・転職検討中の女性

### キャラクター

- 一人称：「僕」
- プロ個人のスカウトマン。論理的・経験豊富・結果重視。
- 夜職女性の味方。稼ぎたい・困っているなら、ロジックで応える人。

### 禁止テーマ（ACCOUNT_FORBIDDEN_KEYWORDS / seeds.py で管理）

| キーワード/テーマ | 理由 |
|---------|------|
| 代理店・パートナー募集・紹介業 | ターゲットは夜職女性。業者向けは別アカウントで行う |
| スカウト代理店・組織的に稼ぐ | B2B訴求はアカウントコンセプトと不一致 |
| 高収益・ビジネス構造・収益モデル | 情報商材感につながる |
| 稼ぎ方を教えます・ノウハウを共有 | 販売系のトーンはNG |

### 有効カテゴリ（active=TRUE）

- ns_01: 夜職あるある
- ns_02: キャバ嬢のリアル
- ns_03: 店選びノウハウ
- ns_04: 移籍・環境変更
- ns_05: ガルバからキャバへのランクアップ
- ns_06: TikTokライブという選択肢
- ns_07: 相談導線投稿

### 無効カテゴリ（active=FALSE）

- ns_08: 代理店向け投稿（将来のキャンペーン用に保持）

## ポリシー適用フロー

```
生成時  → プロンプトに ACCOUNT_FORBIDDEN_KEYWORDS を注入（reference_based_generator.py）
生成後  → detect_forbidden_keywords でチェック → ヒット時 WAITING_REVIEW（generate_from_jobs.py）
承認時  → social_derivative テキストをチェック → ヒット時 READY 拒否（approve_queue.py）
監視    → check_pipeline_integrity.py の content_theme_guard チェック [WARN]
```
